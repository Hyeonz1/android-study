# Material Theming with Jetpack Compose

## 1. Before you begin
#### Material Design이란?
- 안드로이드를 위한 디자인 시스템으로, 앱 UI를 만들기 위한 가이드 라인을 제공한다.
 
#### 배우는 것
- Material Theming 적용
- custom color palette 적용
- custom font 적용
- custom shapes 적용
- top app bar 적용

## 2. App overview
#### 앱
  - 강아지 리스트 보여주기
  - Material Design 사용

#### Color palette
- 라이트 모드와 다크 모드를 고려하여 컬러 팔레트 설정

#### Typography
- Type Role
- Font
- Font weight
- Size

#### Theme file
- 앱 테마에 대한 정보가 담긴 파일
- color, typography, shape를 통해 정의된다.
